# Customer Purchase Behavior Analysis

## Overview
End-to-end data analytics project analyzing customer purchase behavior using SQL, Python, and Power BI.

## Tools Used
- SQL (SQLite)
- Python (Pandas, NumPy)
- Power BI
- Excel

## Key KPIs
- Total Revenue
- Total Orders
- Average Order Value
- Revenue per Customer
- Top 5 Customers by Revenue

## Dashboard
Interactive Power BI dashboard with filters by date, country, and product.

## Project Structure
- data/: dataset
- sql/: SQL analysis
- powerbi/: Power BI dashboard
- screenshots/: dashboard images

## Insights
- Identified high-value customers contributing majority revenue
- Observed seasonal revenue trends
- Highlighted country-level performance differences
